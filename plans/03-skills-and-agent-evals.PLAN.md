# Plan 03: Skills And Agent Evals

## Goal

Measure the models against realistic coding-agent work, not just short coding tasks and single tool calls.

## Tasks

1. Define a skill-aligned evaluation set for planning, debugging, execution, and verification flows.
2. Add multi-step tool-calling tasks with exact argument validation.
3. Add repo-aware bugfix tasks with executable verification.
4. Record structured pass/fail plus rationale excerpts.
5. Rank models separately for raw speed, coding quality, and agent quality.

## Success criteria

- Models can be compared on realistic agent work.
- Tool reliability is measured beyond single-function calls.
- Recommendations can distinguish "fast", "good coder", and "best agent".
